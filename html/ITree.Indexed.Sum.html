Sums of indexed types

In the context of interaction trees, events can be combined using this sum type.


Sum of type constructors Type -> Type.
sum1 : (Type -> Type) -> (Type -> Type) -> (Type -> Type).

| inl1 (_ : E1 X)
| inr1 (_ : E2 X).
